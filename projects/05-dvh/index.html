DVH (Dicom Viewer Html5)

DVH [Dicom Viewer Html5] is a Cordova-driven Android application engineered to view DICOM images.

It is directly based on the Angular application ng-dicomviewer, allowing it to natively read and render DICOM images directly from the phone's local memory.
