# [NgDicomViewer](https://hrhrprasath.github.io/NgDicomViewer)

NgDicomViewer is an Angular application (Single Page Application) built on top of the open-source Javascript / HTML5 libraries provided by the DWV project.

It is predominantly designed to view DICOM images on an HTML5 canvas within modern browsers, completely eliminating the need for any windowed plugins.
